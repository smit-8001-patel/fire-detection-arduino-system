## 🔹 Overview
This project is a smart fire detection system built using Arduino in Tinkercad. It combines multiple sensors to detect fire hazards and trigger emergency alerts.

## 🔹 Features
- Smoke/gas detection
- Temperature monitoring
- Motion detection
- Light intensity sensing (LDR)
- LCD display for status
- Alarm using LED and buzzer

## 🔹 Components Used
- Arduino Uno
- Gas Sensor
- Temperature Sensor
- PIR Sensor
- LDR
- LCD (16x2)
- LED, Buzzer

## 🔹 Working
The system reads sensor data continuously:
- If gas OR temperature exceeds threshold → ALERT
- LED and buzzer turn ON
- LCD shows warning message
- Otherwise → SAFE state

## 🔹 Tools Used
- Tinkercad
- Arduino

## 🔹 Output
- LCD displays status
- LED and buzzer activate during fire detection
